# Trivia Trek
Welcome to Trivia Trek! This is an interactive trivia game made using Node.js. This file will explain how to run the project, what tools, assumptions, and libraries were used when making it, and what I would add next if I had more time to work on this.

## How to run
### Installing Node
This project was made using Node.js, and uses Vite to bundle the code. Because there is no production environment web server for this project, you will have to install Node.js to run it. I recommend using [the official website](nodejs.org), then use the Prebuilt installer for your operating system and computer processor, using the latest Long Term Support (LTS) version. You can check that node was installed correctly by running `node -v` in the command terminal. 
### Running the Project
To run the project after installing Node, open a command console in this folder and run:
```
npm install
npm run dev
```
This will start up the application. You can tell Vite to open your browser to the URL that is hosting your application by pressing `o`. Once you're done, press `q` to stop Vite from hosting the appllication.
